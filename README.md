# Analyse-et-Optimisation-de-la-structure-de-couts-par-l-EBITDA
Développement d'un moteur de redressement financier sous Python. La solution a permis de simuler une réduction de 37% du point mort et d'identifier le plafond de verre structurel du business model, prouvant mathématiquement la nécessité d'un pivot stratégique.
